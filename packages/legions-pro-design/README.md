# legions-pro-echarts
前端可视化图表业务组件库,这里放置了 pro 系列组件，用于支撑重型组件

## 🖥 浏览器兼容性

- 现代浏览器和 Internet Explorer 11 (with [polyfills](https://stackoverflow.com/questions/57020976/polyfills-in-2019-for-ie11))
- [Electron](https://www.electronjs.org/)
